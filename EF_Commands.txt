
Remember to cd to the chirp.sln and delete the old migrations beforehand


Powershell Command
dotnet ef migrations add InitialMigration --context CheepDbContext --project .\src\Chirp.Infrastructure\Chirp.Infrastructure.csproj --startup-project .\src\Chirp.Web\Chirp.Razor.web.csproj

zsh Command:
-delete entire migrations folder (if theres no important data) And then run the two following commands.

dotnet ef migrations add InitialMigration \
  --context CheepDbContext \
  --project ./src/Chirp.Infrastructure/Chirp.Infrastructure.csproj \
  --startup-project ./src/Chirp.Web/Chirp.Razor.web.csproj

dotnet ef database update \
  --project ./src/Chirp.Infrastructure/Chirp.Infrastructure.csproj \
  --startup-project ./src/Chirp.Web/Chirp.Razor.web.csproj
